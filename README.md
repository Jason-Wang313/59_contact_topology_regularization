# Contact Topology Regularization

Submission-hardening v2 artifact for paper 59 in the robotics 60-paper batch.

## Decision

Workshop-only.

The mechanism is interesting, but the evidence is still toy-only. V2 hardening shows both sides of the claim: smoothness-only objectives can miss contact topology, but fixed topology regularization can also fail when a task requires a legitimate topology switch.

## V2 hardening result

The topology-switch stress evaluates three target contact classes: upper, lower, and switching. Smoothness-only and fixed-upper topology each reach 0.333 success and 0.000 switch-task success. Task-conditioned topology reaches 1.000 success only once `lambda_top=1.25`, and 20% topology-label noise drops success to 0.800.
